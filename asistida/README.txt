CMS E-commerce - Bootstrap (Conversión desde Tailwind)
--------------------------------------------------------

Archivos incluidos:
- index.html    : Estructura principal (Bootstrap 5.3.3 vía CDN)
- styles.css    : Estilos personalizados con los colores exactos indicados
- main.js       : Lógica JS (tabs, sidebar, modales, tablas de ejemplo)
- README.txt    : Este archivo

Instrucciones rápidas:
1. Descomprime la carpeta `cms_ecommerce_2025.zip` en tu servidor local o carpeta de trabajo.
2. Abre `index.html` en el navegador (puedes usar Live Server o abrir el archivo directamente).
3. El sidebar es responsive: en pantallas pequeñas aparecerá colapsado con un botón hamburguesa.
4. Las tablas contienen datos simulados. Las acciones de exportar/filtrar son simuladas (alert).

Notas:
- Bootstrap y FontAwesome se cargan por CDN (necesitas conexión a Internet).
- Fuente Inter se carga desde Google Fonts.
- No se incluyó favicon como solicitaste.
- Animaciones: transiciones suaves de 0.4s para cambios de pestaña y modales.
